# 🔐 密室逃脫謎題庫 (Escape Puzzle Library)

一個收集並整理了多年來（106年、108年、110年、112年、113年等）各項密室逃脫活動謎題的數位圖書館。

## 🌟 專案特點

-   **完整索引**：透過 `index.html` 提供強大的搜尋與篩選功能，按類型（語文、密碼、推理等）及活動年份分類。
-   **數位驗證**：提供互動式的謎題驗證網頁（位於 `validators/`），讓玩家可以直接在網路上體驗解謎樂趣。
-   **多格式支援**：包含 PDF 檔、試算表、GeoGebra 檔等多種形式的謎題資源。
-   **自定義美化**：整體風格統一，參考 `樣式設定.md` 進行現代化網頁設計。

## 📁 資料夾結構

-   `index.html`: 謎題庫檢索主介面。
-   `validators/`: 存放互動式謎題驗證網頁。
-   `pdf/`: 存放所有謎題的 PDF 掃描檔或電子檔。
-   `密室逃脫謎題分類整理.md`: 謎題內容與答案的詳細整理清單。
-   `樣式設定.md`: 專案的色彩規範、字體與組件美化指南。

## 🚀 如何開始

1.  直接開啟 `index.html` 即可開始瀏覽所有謎題。
2.  使用搜尋框可以快速找到特定的謎題。
3.  點選卡片中的「🕹️ 點我玩互動驗證」可開啟互動版謎題。
