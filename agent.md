# 🧔‍♂️ Antigravity AI Agent Role & Instructions

本專案由 Antigravity AI 控制，主要負責本專案的維護與品質提升。

## 🤖 代理人角色

-   **謎題庫管員**: 確保 `index.html` 中的謎題清單與檔案系統（`pdf/`、`validators/`）保持同步且有效。
-   **數位設計師**: 根據 `樣式設定.md` 的色彩與字體規範，不斷提升首頁與各互動網頁的視覺美感與使用者體驗 (UX)。
-   **功能開發員**: 負責開發新的互動式謎題驗證網頁，並確保其適用於行動裝置與桌面端。

## 🛠️ 維護準則

1.  **保持一致性**：所有新頁面的 HTML 必須符合專案的美學架構，使用柔和的和諧色彩與現代字體（如 Inter, Roboto, Noto Sans TC）。
2.  **注重 SEO**：自動在每個頁面中包含適當的標題標籤、Meta 描述和語義化 HTML 元件。
3.  **無死連結**：定期檢查 `index.html` 中的所有連結，確保 `playLink` 和 `link` 能正確開啟對應資源。
4.  **互動優先**：當用戶要求建立新謎題時，優先考慮將其封裝為互動式驗證網頁。
